# 📚 ことばクイズ

小学生の語彙力アップを目的とした、日本語学習クイズアプリです。

🔗 **公開URL**: https://kotoba-quiz-rjkrkafdobrdappid87m5cd.streamlit.app/

---

## 🎮 アプリの概要

ひらがな・カタカナ・漢字を使った4択クイズです。
絵文字やヒント文を見て、正しい言葉を選んでください。
漢字にはフリガナが付いているので、低学年のお子さんでも安心して遊べます。

---

## ⭐ レベル

| レベル | 対象 | 内容 |
|--------|------|------|
| ⭐ やさしい | 1〜2年生 | ひらがな中心・絵文字あり |
| ⭐⭐ ふつう | 2〜3年生 | カタカナ・漢字まじり・絵文字あり |
| ⭐⭐⭐ むずかしい | 3〜4年生 | 漢字中心・説明文のみ |

---

## 📂 カテゴリー

どうぶつ／しぜん／たべもの／のりもの／はたらき／スポーツ／どうぐ／ちり

---

## 🛠️ 使用技術

- Python 3.x
- [Streamlit](https://streamlit.io/)

---

## 📁 ファイル構成

```
kotoba-quiz/
├── app.py           # メインアプリ（画面制御・ゲームロジック）
├── questions.py     # 問題データ（全レベル・全カテゴリー）
└── requirements.txt # 依存パッケージ
```

---

## 🚀 ローカルでの起動方法

```bash
# リポジトリをクローン
git clone https://github.com/camelterra-glitch/kotoba-quiz.git
cd kotoba-quiz

# パッケージをインストール
pip install -r requirements.txt

# アプリを起動
streamlit run app.py
```

---

## ✨ 主な機能

- 3段階のレベル選択
- 問題・選択肢のランダム表示
- 漢字へのフリガナ表示（ヒント文・選択肢）
- 正誤フィードバック（正解・不正解のハイライト）
- 進捗バー・スコア表示
- 結果画面での全問振り返り
- クイズ中にトップ画面へ戻るボタン

---

## 📝 問題を追加するには

`questions.py` の `QUESTIONS` リストに辞書を追加するだけです。

```python
{
    "emoji": "🐶",
    "hint": "「ワンワン」と なく、いえで かう どうぶつは？",
    "hint_ruby": "「ワンワン」と なく、いえで かう どうぶつは？",
    "answer": "いぬ",
    "choices": ["いぬ", "ねこ", "うさぎ", "くま"],
    "choices_ruby": {},
    "level": "easy",       # "easy" / "normal" / "hard"
    "category": "どうぶつ",
},
```
